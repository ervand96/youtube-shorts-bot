# ABC Song for Toddlers

## Topic
- Hook: Benny taps alphabet blocks and sings the ABCs in order
- Age: 2-8
- Category: learning
- Moral: Learning feels fun when we try one letter at a time
- Visual: Bright alphabet blocks, Benny pointer, bouncing letters A B C

## Script
TAP TAP! Benny stacks A, B, C up high!
A is for apple, red and sweet.
B is for bunny with bouncy feet.
C is for car that goes vroom today.
D is for duck who waddles away.
Letters can dance, letters can sing.
Try one more sound, then clap for everything.
TAP TAP! Great job learning ABCs with Benny!
